# Design System

## Required Fonts

Fonts are not tracked in version control and must be installed manually into the `fonts/` directory.

### Funnel

Clone into `fonts/`:

```bash
git clone https://github.com/Dicotype/Funnel.git fonts/Funnel
```

Then move the font folders from `fonts/Funnel/fonts/` into `fonts/`:

- `Funnel_1001`
- `Funnel_Display`
- `Funnel_Sans`
